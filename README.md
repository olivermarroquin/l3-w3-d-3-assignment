# Posts Search App

## What the app does

This React app fetches post data from the DummyJSON API and displays the posts on the page. The user can type in the search input to filter posts by title. The reset button clears the search and shows all posts again.

## How to run the app

1. Install dependencies:

```bash
npm install
```

2. Start the app:

```bash
npm run dev
```

3. Open the local link shown in the terminal.

## API used

This app uses:

https://dummyjson.com/posts

## Features

Fetches data with useEffect
Stores API data in state with useState
Renders posts on the page
Lets the user search posts by title
Includes a reset button
